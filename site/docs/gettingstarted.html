Getting started with Connect

The Connect documentation will aid application developers, API owners, site administrators and end-users on how to use the platform.

Connect is an API Platform for and higher- and lower-education and research in Norway. Connect allows service providers to access a wide variety of APIs and data sets on top of a robust and standard compliant OAuth API base layer.
