OpenID Connect

Implementation in progress

If you are interested in playing with OpenID Connect in the pilot, contact us about this.

We plan on making OpenID Connect support publicly available during the pilot.

Discovery and configuration

All you need to know in order to configure your OpenID Connect client to the Connect platform, is available through the discovery endpoint:

  • https://auth.feideconnect.no/.well-known/openid-configuration

OpenID Specifications

  • OpenID Connect Core OpenID Connect Discovery

Supported features

  • Authorization Code flow (response_mode: code)
  • Implicit grant flow (response_mode: token)
  • IDtoken signed with PKI (RS256)

Dynamic registration is (not yet) supported.

Authentication

Connect will enable OpenID Connect functionality on the same authorization endpoint as the one used with basic OAuth, at the moment the client is requesting a token with the scope openid.
