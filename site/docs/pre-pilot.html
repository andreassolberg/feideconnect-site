Getting started with Service Providers / Applications

As an service provider, you may register a client which implement OAuth Consumer functionality.

Learn about OAuth before moving on.

Register a client using the Developer Dashboard. When registering a client, you need to know the OAuth redirect URI endpoint of your application.

HTTP or HTTPS The Connect pilot will allow you to register Redirect URIs that are not secured with HTTPs, although not reccomended. A production enviornment with Connect, would probably forbid registering unsecured endpoints.
