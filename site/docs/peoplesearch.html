People search

People search is an API that allows you to perform text search for end users within an organization.

As of now, the lack of user identifiers returned in the API is limiting how it can be used. The People Search API is used within the groups app to manage ad-hoc groups.

We are interesting in discussing a proper way of releasing user identifiers in this API with organizations.
