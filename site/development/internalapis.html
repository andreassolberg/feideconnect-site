Interne APIer i Connect plattformen.

Client Administration API

documentation

  • https://api.feideconnect.no/clientadm/clients/ (POST) Create a client
  • https://api.feideconnect.no/clientadm/clients/9dd084a3-c497-4d4c-9832-a5096371a4c9 (PATCH) Update a client
  • https://api.feideconnect.no/clientadm/clients/9dd084a3-c497-4d4c-9832-a5096371a4c9/scopes (PATCH) Update scopes for a client
  • https://api.feideconnect.no/clientadm/clients/9dd084a3-c497-4d4c-9832-a5096371a4c9 (GET) Info about client (can also be unauthenticated)
  • https://api.feideconnect.no/clientadm/clients/ (GET) List all clients by a user
  • https://api.feideconnect.no/clientadm/clients/?organization={org id} (GET) List all clients by an org
  • https://api.feideconnect.no/clientadm/clients/?scope=clientadmin (GET) Filtering list by scope
  • https://api.feideconnect.no/clientadm/public (GET) Public information about all clients
  • https://api.feideconnect.no/clientadm/clients/9dd084a3-c497-4d4c-9832-a5096371a4c9 (DELETE) Delete client
  • https://api.feideconnect.no/clientadm/clients/9dd084a3-c497-4d4c-9832-a5096371a4c9/logo Client logo
  • https://api.feideconnect.no/clientadm/scopes/ (GET) List public scope definitions

Ad Hoc Group Administration API

documentation

  • https://api.feideconnect.no/adhocgroups/ (POST) Create a group
  • https://api.feideconnect.no/adhocgroups/{group id} (PATCH) Update group
  • https://api.feideconnect.no/adhocgroups/{group id} (DELETE) Delete group
  • https://api.feideconnect.no/adhocgroups/{group id} (GET) View group info
  • https://api.feideconnect.no/adhocgroups/{group id}/details (GET) View group details
  • https://api.feideconnect.no/adhocgroups/{group id}/members (GET) View group members
  • https://api.feideconnect.no/adhocgroups/{group id}/members (PATCH) Update group members
  • https://api.feideconnect.no/adhocgroups/{group id}/members (DELETE) Update group members
  • https://api.feideconnect.no/adhocgroups/memberships (GET) List memberships
  • https://api.feideconnect.no/adhocgroups/memberships (DELETE) Delete memberships
  • https://api.feideconnect.no/adhocgroups/memberships (PATCH) Confirm memberships
  • https://api.feideconnect.no/adhocgroups/{group id}/invitaiton (POST) Join group using invitation token
  • https://api.feideconnect.no/adhocgroups/{group id}/logo Group logo
