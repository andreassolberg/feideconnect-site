Feide Connect Auth Engine

Kildekode Feide Connect Auth Engine – påbegynt desember 2014

Denne komponenten utvikles på auth.dev.feideconnect.no i master-branchen.

En stabil versjon med hyppige inkrementelle oppdateringer, som etterhvert skal bli komponenten som benyttes i piloten 2015, finnes i pilot branchen i det samme koderepositoriet.

Per desember 2014 er denne kun satt opp å kunne autentisere med Feide. Vi vil her koble inn edugain, kalmar, IDporten og gjeste IdP løsning etterhvert.

Pilot instansen av Feide Connect Auth Engine kjører på auth.feideconnect.no.

Storage - Cassandra

Piloten kjører mot cassandra clusteret, med keyspacet feideconnect. Dette vil være populert med endel tilfeldig genererte brukere og klienter.

API for utstedelse av tokens (til testformål)

Dette API endepunktet vil dumpe ut en access token til testformål, med litt debug info, i tekstformat.

Endepunkt:

{% highlight plaintext %} https://auth.feideconnect.no/poc/{user}/{client} {% endhighlight %}

Parameteret {user} kan være:

@me
En autentisert sluttbruker (vil kreve pålogging via Feide)
03c01b62-001a-40d7-bb62-c0a5bb90d5bd
Bruker UUID for en vilkårlig bruker.
@random
En tilfeldig testbruker. En ny bruker velges for hver forespørsel.
@none
Ingen bruker. Bruk denne for å simulere tokens som er utstedt til klienter vha client credentials flow. Slike tokens kan kun representere en autentisert klient og ingen spesielle brukere.

Parameteret {client} kan være:

af6386b5-d131-4651-a90a-f33299a8df60
Klient UUID for en vilkårlig klient.
@random
En tilfeldig testklient. En ny klient velges for hver forespørsel.

Eksempelkall:

  • https://auth.feideconnect.no/poc/@me/@random
  • https://auth.feideconnect.no/poc/@random/@random
  • https://auth.feideconnect.no/poc/904cec02-a893-436c-95a4-da986c6c53ea/86d64772-9be2-475f-9062-151a081e02b5
  • https://auth.feideconnect.no/poc/@random/86d64772-9be2-475f-9062-151a081e02b5
  • https://auth.feideconnect.no/poc/@none/@random
